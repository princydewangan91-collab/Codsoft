# Titanic Survival Prediction

This project is a part of my CodSoft Machine Learning Internship. The objective is to predict whether a passenger survived the Titanic disaster using machine learning techniques.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Feature encoding using Pandas
* Train-test split for model evaluation
* Random Forest Classifier implementation
* Feature importance analysis

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Dataset

The Titanic dataset contains passenger information such as:

* Passenger Class (Pclass)
* Gender
* Age
* Fare
* Number of Siblings/Spouses
* Number of Parents/Children
* Embarked Port

## Results

The model was trained using the Random Forest algorithm and evaluated on the test dataset to predict passenger survival.

This project was completed as Task 1 of the CodSoft Machine Learning Internship.

# Movie Rating Prediction

This project is a part of my CodSoft Machine Learning Internship. The objective is to predict movie ratings using machine learning techniques based on features such as genre, director, actors, duration, year, and votes.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Feature encoding using Label Encoder
* Feature engineering and data transformation
* Train-test split for model evaluation
* Random Forest Regressor implementation
* Feature importance analysis
* Rating prediction and visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

The IMDb India Movies dataset contains movie information such as:

* Genre
* Director
* Actor 1
* Actor 2
* Actor 3
* Duration
* Year
* Votes
* Rating

## Results

The model was trained using the Random Forest Regression algorithm and evaluated using MAE, RMSE, and R² Score to predict movie ratings accurately.

This project was completed as Task 2 of the CodSoft Machine Learning Internship.
