# Titanic Survival Prediction

This project is a part of my CodSoft Machine Learning Internship. The objective is to predict whether a passenger survived the Titanic disaster using machine learning techniques.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Feature encoding using Pandas
* Train-test split for model evaluation
* Random Forest Classifier implementation
* Feature importance analysis

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Dataset

The Titanic dataset contains passenger information such as:

* Passenger Class (Pclass)
* Gender
* Age
* Fare
* Number of Siblings/Spouses
* Number of Parents/Children
* Embarked Port

## Results

The model was trained using the Random Forest algorithm and evaluated on the test dataset to predict passenger survival.
